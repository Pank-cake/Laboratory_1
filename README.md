# Задание 1. Генерация списков

## Задача 1

### Текст задачи

Сформировать список из значений 1 и 0 по принципу: 0 — если очередное введенное число 
нечетное, 1 — в противном случае. 

### Алгоритм решения

1) Запрашиваем у пользователя количество чисел, которое он будет вводить.
2) С помощью функции проверяем, чтобы количество, которое он запросил, было натуральным числом.
3) Запршиваем число с пользователя (проверяем функцией на натуральность).
4) Если остаток от деления этого числа на 2 равен 0, то число чётное и записываем в список 1, 
если остаток от деления не равен 0, то число нечётное - записываем 0.
5) Повторяем шаги 3 и 4 до тех пор, пока счётчик не сравняется с количеством чисел, которое пользователь хотел ввести.
6) Выводим получившийся список.

# Задание 2. Рекурсия

## Задача 1

### Текст задачи

Сформировать список из четных цифр числа. 

### Алгоритм решения

1) Запршиваем с пользователя число, из цифр которого будем формировать список.
2) С помощью функции проверяем, чтобы число, которое пользователь запросил, было натуральным.
3) Рекурсивной функцией на возврате проверяем на последнюю цифру числа на чётность, если цифра чётная - записываем в начало списка и возвращаем оставшуюся часть числа,
если нечётная - просто возвращаем оставшуюся часть числа.
4) Проверяем рекурсивно все цифры числа на чётность до того момента, пока число не станет 0.
5) Выводим получившийся список.

# Задание 3. 

## Задача 1

### Текст задачи

Опишите функции, осуществляющие основные математические операции над комплексными 
числами (сложение, вычитание, умножение, деление, возведение в степень). Использовать 
готовый класс Complex запрещено. 

### Алгоритм решения

1) Запрашиваем у пользователя два комплексных числа: действительную и мнимую часть одного, затем второго. 
2) С помощью функции считываем числа и проверяем, чтобы числа, которые пользователь ввёл, были целыми.
3) Запрашиваем у пользователя операцию 1-5, которую требуется выполнить над числами.
4) С помощью функции проверяем, чтобы операция существовала.
5) Если операция сложения вызываем функцию сложения, а в ней:
	- Складываем дейтсвительную часть одного с действительной частью другого, мнимую часть одного с мнимой частью другого.
	- Результат сложения записываем и выводим с помощью функции.
   Если операция вычитания, вызываем функцию вычитания, а в ней:
	- Вычитаем дейтсвительную часть одного из действительной части другого, мнимую часть одного из мнимой части другого.
	- Результат вычитания записываем и выводим с помощью функции.
   Если операция умножения, вызываем функцию умножения, а в ней:
	- Перемножаем действительную часть одного на действительную часть другого, вычитаем произведение мнимых частей для получения действительной части,
	к произведению действительной части одного на мнимую часть другого суммируем произведение действительной части первого на миную второго для получения мнимой части.
	- Результат произведения записываем и выводим с помощью функции.
   Если операция деления, вызываем функцию деления, а в ней:
	- Вычисляем знаменатель формулы - суммируем кведрат действительной части второго числа и квадрат его мнимой части.
	- Проверяем, чтобы знаменатель не был равен 0, если равен - выдаём ошибку и снимаем программу с выполнения.
	- Вычисляем действительную часть - сумму произведения действительных частей с произведением мнимых делим на вычисленный знаменатель
        и мнимую - разность произведения мнимой первого на действительную второго с произведением действительной первого на мнимую второго делим на знаменатель.
	- Результат деления записываем и выводим с помощью функции.
   Если операция возведения в степень, вызываем функцию возведения в степень, а в ней:
	- Запрашиваем степень у пользователя и проверяем с помощью функции, чтобы она была целым положительным числом.
	- Если степень 0 - записываем число 1 + 0i и выводим его с помощью функции.
	- Если степень больше 0 - повторяем циклом for раз в заыисимости от степени:
		--Умножить текущее (изначальное) значение (1 + 0i) на исходное число
		--Пересохранить результат
	- Возвращаем результат и выводим с помощью функции.
